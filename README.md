# NYC Yellow Taxi Dashboard (Jan 2024)

This project builds a data pipeline and interactive dashboard using NYC Yellow Taxi trip data for January 2024.

## Project Components
- Data ingestion from CloudFront
- Data cleaning using Polars
- SQL analysis using DuckDB
- Interactive Streamlit dashboard with filters and visualizations

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

The dashboard will open at:
Local URL: http://localhost:8501
Network URL: http://10.10.20.207:8501
